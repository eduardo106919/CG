## Curvas e Superficies

- [Bezier Curves from the ground up](https://jamie-wong.com/post/bezier-curves/) - A gentle introduction and some use cases
- [Mathematics of Animation Curves](https://www.khanacademy.org/computing/pixar/animate/parametric-curves/a/lesson-brief-animation) - An introduction focused on animation
- [A Primer on Bézier Curves](https://pomax.github.io/bezierinfo/) - All the math required (sections: 1-4, 6-8, 34-36)
- [Catmull-Rom Splines in Game Development](https://andrewhungblog.wordpress.com/2017/03/03/catmull-rom-splines-in-plain-english/)
- [Catmull-Rom Splines](https://www.cs.cmu.edu/~462/www/projects/assn2/assn2/catmullRom.pdf)
- [Smooth Paths Using Catmull-Rom Splines](https://qroph.github.io/2018/07/30/smooth-paths-using-catmull-rom-splines.html) - Shows both code and maths
- [Black Pawn](https://blackpawn.com/texts/splines/) - Interactive site with some cubic curves
